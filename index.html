Estrutura básica de um documento HTML5

Voltar

As tags HTML5 são elementos utilizados para estruturar e organizar o conteúdo de páginas web. Elas trazem melhorias significativas em relação às versões anteriores do HTML, com novos recursos voltados para multimídia, semântica e interatividade. Aqui está uma lista das principais tags introduzidas ou aprimoradas no HTML5:

Estrutura Semântica

Voltar

<header>: Define o cabeçalho de uma página ou seção.
<nav> : Representa uma seção de links de navegação.
<section>: Agrupa conteúdos tematicamente relacionados.
<article>: Representa um conteúdo independente e autossuficiente (artigos, posts).
<aside>: Conteúdo relacionado, mas não essencial (barra lateral, notas).
<footer>: Define o rodapé da página ou seção.
<main>: Indica o conteúdo principal da página.

Multimídia

Voltar

<audio>Reproduz arquivos de áudio.
<vídeo>Reproduz arquivos de vídeo.
<source: >Especifica várias fontes para <audio>ou <vídeo>
<track>Adiciona legendas ou descrições em vídeos ou áudios.

Formulários e Interatividade

Voltar

<form>Contém elementos de entrada e envio de dados.
<input>>: Campo para entrada de dados (melhorado com novos tipos como email, date, range).
<textarea>Área para texto multilinha
<button>Botão clicável.
<datalist>Lista de opções para um campo <input>.
<output>Mostra o resultado de um cálculo. <meter>Representa um valor dentro de um intervalo.
<progress>Barra de progresso.

Gráficos e Animação

Voltar

<canvas>: Desenha gráficos 2D ou 3D com JavaScript.
<svg>:Define gráficos vetoriais escaláveis.

Tags Diversas

<figcaption>: Adiciona uma legenda a < figure >
< time >Representa datas ou horários.
<mark>: Destaca texto relevante.
<summary>: Cabeçalho clicável para o elemento <details>. <details>: Exibe ou oculta conteúdo adicional.

API e Integrações

<template>: Contém conteúdo que pode ser reutilizado via JavaScript.
<slot>: Define pontos de inserção para conteúdo em Web Components.

Essas tags tornam o HTML5 mais poderoso, com foco na acessibilidade e no suporte a dispositivos modernos.
